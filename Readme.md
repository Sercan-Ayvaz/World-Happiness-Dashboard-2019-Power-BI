# World Happiness Dashboard (2019) | Power BI



## Amaç

Bu çalışma, *World Happiness Report 2019* verilerini analiz ederek ülkelerin mutluluk seviyelerini ve bunu etkileyen sosyo-ekonomik faktörleri görselleştirmeyi amaçlar.

## Veri Seti

- Kaynak: World Happiness Report (2019)
- Ülke Sayısı: 156
- Yıl: 2019
- Metrikler:
&nbsp; - Mutluluk Skoru
&nbsp; - Kişi Başı GSYİH
&nbsp; - Özgürlük
&nbsp; - Sağlıklı Yaşam Beklentisi
&nbsp; - Sosyal Yardım
&nbsp; - Cömertlik
&nbsp; - Yolsuzluk Algısı

## Dashboard İçgörüleri
- En mutlu ülke: **Finland**
- Global ortalama skor: **5.41**
- Ekonomik refah ile mutluluk arasında **pozitif korelasyon** gözlendi
- En yüksek ortalamaya sahip bölgeler: **Okyanusya** ve **Avrupa**



| Bölge | Ortalama Skor |
|---|---|
| Okyanusya | 7.27 |
| Avrupa | 6.28 |
| Kuzey Amerika | 6.15 |
| Güney Amerika | 5.92 |
| Asya | 5.26 |
| Afrika | 4.37 |



Bu analiz, ekonomik göstergelerin önemli bir rol oynadığını ancak mutluluğun çok faktörlü bir olgu olduğunu göstermektedir.

## Nasıl Kullanılır?

1. Repo içindeki `.pbix` dosyasını indiriniz
2. Power BI Desktop ile açınız
3. Rapor içindeki filtre ve segmentleri kullanarak etkileşimli analiz yapabilirsiniz
4. Veri kaynaklarını güncellemek için: `Transform Data / Edit Queries` bölümünü kullanabilirsiniz
